# PythonShell

A simple shell created using python.

### Supports

- DIR
- CD
- CLS
- EXIT
- Redirection (Using the '>' Operator)
- Piping (Using the '|' Operator)

### Instructions

In order to start the shell run the `shell.py` python script.
